# jameszedd.com

Personal site for James Zedd — a single-page professional CV with highlights from his martial arts training.

Built with [Next.js 16](https://nextjs.org) (App Router) and [Tailwind CSS v4](https://tailwindcss.com).

## Page sections

1. Introduction
2. Summary of experience
3. Career highlights — The Interactive Abode
4. Career highlights — Halight Inc.
5. Portfolio projects
6. Contact

## Development

Requires Node 22 (see `.nvmrc`).

```bash
nvm use
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
